# Production Readiness Checklist

## Business Checklist

- Problem statement approved.
- In-scope and out-of-scope behavior approved.
- Product owner assigned.

## Data Checklist

- Knowledge base approved.
- Owners assigned.
- Conflicts resolved.
- Refresh process documented.

## Evaluation Checklist

- Golden dataset approved.
- Baseline run complete.
- Weak cases documented.
- Acceptance thresholds approved.

## Technical Checklist

- Ingestion works.
- RAG app works.
- Evaluator works.
- Tests pass.
- Docker deployment works.

## Security Checklist

- Secrets not committed.
- Access control design documented.
- Audit requirements documented.
- Incident response path defined.

## MLOps Checklist

- Reindex process documented.
- Rollback process documented.
- Monitoring fields defined.
- Collection, prompt, and dataset versions tracked.

## Go/No-Go Criteria

Go only when critical requirements pass, owners accept residual risks, and rollback is ready.
