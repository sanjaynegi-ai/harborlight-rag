# Data Readiness Checklist

- File formats are approved Markdown.
- Metadata can be derived from path and filename.
- Every document has an owner.
- Every document has an approval status.
- Document freshness is known.
- Deprecated files are removed or labeled.
- Duplicate content is resolved.
- Conflicting facts are reconciled.
- Missing critical documents are tracked.
- Confidentiality level is known.
- Refresh cadence is documented.

## Quality Gate Before Ingestion

Ingestion begins only when approved documents are present, owners are known, conflicts are resolved, and missing documents are accepted as known gaps.
